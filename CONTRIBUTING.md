# Contributing to Gid Paragon Hub

Thank you for helping make this repository more useful for students! 🎉

## How to contribute

1. Fork the repository
2. Create a feature branch
3. Make your update (resource, guide improvement, typo fix, etc.)
4. Commit with a clear message
5. Open a pull request

## Contribution ideas

- Add verified free learning resources
- Improve roadmap clarity
- Fix broken links
- Add beginner-friendly explanations
- Suggest monthly challenge templates

## Pull request checklist

- [ ] I explained why this change is useful
- [ ] I checked links and formatting
- [ ] I kept explanations beginner-friendly
- [ ] I followed respectful community behavior

## Community behavior

Be kind, constructive, and inclusive. Help others learn.
