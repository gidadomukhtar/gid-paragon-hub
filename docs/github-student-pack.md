# 🎓 GitHub Student Developer Pack

The **GitHub Student Developer Pack** gives students free access to premium developer tools and services.

## Why this is useful

- Helps students access paid tools for free
- Supports project development, learning, and collaboration
- Great for building your portfolio with real tools

## Core Resources

| Resource | Link | Why it matters |
|---|---|---|
| GitHub Student Developer Pack | https://education.github.com/pack | Free tools and credits for students |
| GitHub Education | https://education.github.com | Programs, events, and student opportunities |
| GitHub Global Campus | https://education.github.com/globalcampus | Global student tech community |
| GitHub Campus Experts | https://education.github.com/experts | Leadership pathway for student community builders |

## How to apply

1. Create a GitHub account
2. Verify your student status
3. Apply at the Student Pack page
4. Explore available offers and activate what you need

> [!TIP]
> Use your Student Pack tools to build projects you can showcase in interviews.
