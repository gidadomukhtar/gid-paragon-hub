# 📘 Microsoft Learn

**Microsoft Learn** is a free, interactive learning platform for cloud, AI, data, security, and software development.

## Why this is useful

- Structured learning paths for beginners and advanced learners
- Hands-on labs and practical modules
- Skill badges to validate progress

## Core Microsoft Learning Resources

| Resource | Link | Why it matters |
|---|---|---|
| Microsoft Learn | https://learn.microsoft.com/training/ | Free role-based learning paths |
| Azure for Students | https://azure.microsoft.com/free/students/ | Cloud credits and free services for students |
| AI Skills Navigator | https://aiskillsnavigator.microsoft.com | Helps choose AI learning journey by role |
| Microsoft Applied Skills | https://learn.microsoft.com/credentials/applied-skills/ | Practical scenario-based skill credentials |
| Microsoft Certifications | https://learn.microsoft.com/credentials/certifications/ | Industry-recognized role-based certifications |
| Microsoft Reactor | https://developer.microsoft.com/reactor/ | Free events, workshops, and community learning |
| Microsoft for Startups Founders Hub | https://foundershub.startups.microsoft.com | Startup support, tools, and cloud resources |

## Suggested learning path

1. Start with Git and GitHub basics
2. Learn cloud fundamentals (Azure)
3. Choose a role: Developer, Data, AI, Security
4. Build projects and earn applied skills badges
