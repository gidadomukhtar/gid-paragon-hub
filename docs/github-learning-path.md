# 🧭 GitHub Learning Path

A practical roadmap for learning Git and GitHub from beginner to confident contributor.

## Why this is useful

GitHub is essential for collaboration, portfolio building, and open-source contribution.

## Essential GitHub Resources

| Resource | Link | Why it matters |
|---|---|---|
| GitHub Skills | https://skills.github.com | Interactive, beginner-friendly GitHub courses |
| GitHub Docs | https://docs.github.com | Official guides and best practices |
| GitHub Education | https://education.github.com | Student programs and opportunities |
| GitHub CLI | https://cli.github.com | Work with GitHub from terminal |
| GitHub Desktop | https://desktop.github.com | Simple visual Git workflow |
| GitHub Learning Lab alternatives | https://skills.github.com | Skills replaces Learning Lab with modern courses |

## 4-week plan

### Week 1: Foundations
- Repositories, commits, branches
- Markdown basics
- Pull requests and code review concepts

### Week 2: Team workflow
- Forking and branching strategy
- Issue tracking and project planning
- Writing meaningful commit messages

### Week 3: Collaboration
- Review PRs
- Resolve merge conflicts
- Improve documentation

### Week 4: Contribution
- Choose a beginner-friendly repo
- Submit first meaningful PR
- Reflect and improve
