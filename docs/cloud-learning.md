# ☁️ Cloud Learning

Cloud computing skills are in high demand and useful across software engineering roles.

## Why this is useful

- Helps you deploy and scale applications
- Introduces modern DevOps workflows
- Improves employability for internships and jobs

## Recommended Resources

| Resource | Link | Why it matters |
|---|---|---|
| Microsoft Learn (Azure) | https://learn.microsoft.com/training/azure/ | Structured Azure learning paths |
| Azure for Students | https://azure.microsoft.com/free/students/ | Free credits for hands-on cloud projects |
| Google Developers | https://developers.google.com | Practical cloud and platform docs |
| AWS Skill Builder (free tier resources) | https://skillbuilder.aws | Cloud fundamentals and labs |
| Cisco Networking Academy | https://www.netacad.com | Networking and infrastructure foundations |

## Beginner cloud path

1. Cloud fundamentals (compute, storage, networking)
2. Deploy static and dynamic apps
3. Learn CI/CD basics
4. Add monitoring and security practices
