# 🛣️ Learning Roadmap

Curated paths for students to grow from beginner to job-ready.

## 1) Git & GitHub
- Git basics, commits, branches, PRs
- Issue tracking and project boards
- Team collaboration workflow

## 2) Web Development
- HTML, CSS, JavaScript
- Responsive design
- Frontend frameworks (React/Vue)

## 3) Backend Development
- Server fundamentals
- APIs (REST/GraphQL)
- Databases and authentication

## 4) Python
- Core syntax and problem solving
- Scripting and automation
- Web/data/AI ecosystem exploration

## 5) C#
- Language fundamentals
- OOP and application structure
- Tooling and debugging

## 6) .NET
- ASP.NET Core APIs
- Entity Framework basics
- Deployment and monitoring

## 7) Cloud Computing
- Cloud fundamentals
- Deployments and CI/CD
- Security and cost awareness

## 8) AI
- Python + data basics
- Intro ML workflows
- Build simple AI-powered applications

## 9) Cybersecurity
- Security fundamentals
- Networking basics
- Threat awareness and secure coding
